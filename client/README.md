# Event Management System (Frontend)

This is the frontend for the Event Management System, built with React (MERN Stack).

## Tech Stack
- **React**: Frontend library
- **Vite**: Build tool
- **React Router**: Navigation
- **Lucide React**: Icons
- **CSS3**: Modern styling with Flexbox/Grid and Variables

## Features
- **Responsive Navbar**: With mobile menu support.
- **Hero Section**: Animated entrance and gradient typography.
- **Event Listing**: Grid layout with hover effects on cards.
- **Event Card**: Displays image, date, location, and description.

## Getting Started

1. Navigate to the client directory:
   ```bash
   cd client
   ```

2. Install dependencies (if not already done):
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser at `http://localhost:5173`.
